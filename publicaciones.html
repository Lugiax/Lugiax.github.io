Uso de Algoritmos Genéticos y Redes Neuronales para Control Predictivo en un Reactor tipo CSTR
Tesis de licenciatura - UNAM

Diseño de un sistema de control predictivo para la operación de un CSTR para la producción de propilenglicol, capaz de reaccionar a diferentes tipos de perturbaciones. El sistema utiliza una red neuronal para estimar los parámetros ideales de control, que fueron encontrados por medio de un algoritmo genetico. Tanto la red como el algoritmo genetico fueron programados desde cero con Python. Liga a la tesis.

Registro de Nubes de Puntos 2D: Comparación entre Métodos Iterativos y una Red neuronal Artificial
CI2T 2021

El trabajo se basó en la comparación de la eficientia entre tres algoritmos iterativos (Iterative Closest Points, Random Sample Consensus y Coherent Point Drift) y una arquitectura propia de Red Neuronal Artificial, para la tarea de registro de nubes de puntos en 2D.
